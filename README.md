# Eshop M-Commerce App

## Description
Eshop M-Commerce App is an Android-based mobile commerce application for selling laptops and electronics.  
It provides a smooth shopping experience for customers while allowing admins to efficiently manage products.

---

## Technologies Used
- **Java** – Core programming language for app logic  
- **XML Layouts** – UI design and layout management  
- **Firebase Authentication** – Secure user registration and login  
- **Firebase Firestore / Realtime Database** – Data storage and retrieval  
- **MySQL** – Backend database for certain modules  
- **Material Design Components** – Modern UI design elements  

---

## Current Completed Features
- **User Sign Up / Login** – Secure authentication system  
- **Category Listing** – Display products by categories  
- **Home Slide Bar** – Featured products and promotions  
- **Bottom Navigation Bar** – Easy navigation between sections  
- **Product Listing** – View all available products  
- **Single Product View** – Detailed view of each product  

---

## Target Users
- **Customers** – Can browse and purchase products  
- **Admins** – Can add, update, or manage products  

---

## Future / Pending Features
- Cart System  
- Checkout & Payment Integration  
- Admin Panel Enhancements
